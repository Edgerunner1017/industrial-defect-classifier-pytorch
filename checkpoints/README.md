# Checkpoints Guide

This directory is reserved for trained model checkpoints.

Do not commit large model weight files directly to Git.

Local checkpoint examples:

```text
best.pth
last.pth
resnet18_best.pth
efficientnet_b0_best.pth
```

If a trained model needs to be shared publicly, upload it through GitHub Releases or another external storage method and document the download link in the main README.
